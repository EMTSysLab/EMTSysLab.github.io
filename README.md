# EMTSysLab

EMTSysLab is a personal IT lab and knowledge base focused on **Linux administration, IT automation, GLPI (Ticketing), Kubernetes, and related software tools**.

EMTSysLab es un laboratorio y base de conocimientos personales centrado en **administración de Linux, automatización IT, GLPI (Ticketing), Kubernetes y herramientas de software relacionadas**.
